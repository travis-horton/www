# getting started

to get a local version:

### prerequisites

[yarn](https://yarnpkg.com)
[docker](https://www.docker.com)

### setup

make sure you have yarn

### install

yarn

### usage

yarn serve runs parcel serve, which sets up a version of the app on localhost:1234
